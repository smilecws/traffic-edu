# PRD: 초심찾기 도로교통법

## 목표
초심으로 돌아가 도로교통법을 학습하는 어플입니다. 단순 학습뿐 아니라 40분 모의고사, 오답 복습, 즐겨찾기, 문항별 누적 통계로 약점을 스스로 파악할 수 있게 한다.

## 사용자
- 1차: 한국어로 학과시험을 준비하는 국내 응시자
- 2차: 영어/중국어/베트남어 자료로 준비하려는 외국인 응시자 (동일 문제 은행의 번역본 제공)
- 기기: 주로 모바일(Android/iOS), 보조로 웹(GitHub Pages), Windows 데스크톱(WMV 동영상 문제 재생용)

## 핵심 기능

### 1. Google 로그인 + PIPA 동의 게이트
- 앱 첫 실행 시 `ConsentScreen` 에서 Google 로그인 + 이름 입력 + 개인정보 수집·이용 동의(PIPA) 를 완료해야 진입.
- 동의 기록(`ConsentRecord`)은 `SharedPreferences` 에 로컬 저장. 버전 불일치 시 재동의 요청.
- 데스크톱(Windows/macOS/Linux) 은 `google_sign_in` 미지원 → 게이트 우회.
- 홈 우상단 메뉴(점 3개)에서 동의 철회 가능 → `ConsentService.clear()` + `GoogleAuthService.signOut()` → `ConsentScreen` 재표시.

### 2. 접속 로그 (Apps Script Web App)
- 동의 완료(`consent_granted`) 및 앱 실행(`app_launch`) 시 이벤트를 Apps Script Web App 으로 전송.
- Google ID Token 을 매 요청마다 발급해 Apps Script 가 서명·aud·iss·exp 를 검증.
- 전송 실패 시 로컬 큐(`access_log_pending_v1`)에 적재 → 다음 실행 시 flush.
- UI 를 차단하지 않음 (fire-and-forget).

### 3. 홈 랜딩
- 6개 카드 메뉴: **학습하기**, **문제 풀기**, **시험 순서**, **준비 가이드**, **교육 일정**, **시험 일정**.
- 시험 순서·준비 가이드·교육 일정·시험 일정은 `ExamGuideScreen` 의 static 메서드로 도로교통공단 공식 페이지를 `url_launcher` 로 열거나 안내 화면을 표시.

### 4. 학습하기 (StudyCard)
- `StudyScreen` 에서 16개 학습 토픽 목록 표시(속도와 차로 통행 / 음주·약물 운전 규제 / 이륜차·PM 등).
- 각 토픽을 탭하면 `StudyCardScreen` 에서 3개 세부 주제(A/B/C) 아코디언과 그 안의 5장 카드뉴스(뱃지·본문·핵심 포인트·비교 표·태그)를 가로 스와이프로 학습.
- 하단에 "시험 출제 분석" 섹션으로 빈출 출제 포인트를 정리.
- 학습 카드 데이터: `assets/study/NN_<slug>.json` (NN=01~16, 사람이 직접 작성, 한국어 단일 언어, 스키마는 `StudyTopic` 모델).
- 학습 토픽 축(16개)은 출제 영역 기준 분류 — 학습 화면에서만 사용.

### 5. 4지선다 퀴즈 풀이
- 단일 선택(1개 정답)과 복수 선택(2개 이상 정답)이 한 세션에 섞여 있다. 이미지/동영상 문항 포함. 1,000문제에서 랜덤 40문제 출제가 기본.

### 6. 모의고사 모드 (`showTimerAndScore: true`)
- 40분 타이머, 시험 중 정답 공개 없음, 미답 제출 시 오답 처리.
- 시작 시 면허 종류 선택: 1종 대형 / 1종 특수 / 1종 보통 / 2종 보통.
- 합격 기준: 100점 환산 기준 1종 계열 70점, 2종 보통 60점.
- 결과 기록은 `MockExamHistoryService` 에 최대 80건 보존 (FIFO).

### 7. 카테고리 연습 (`showTimerAndScore: false`, `shuffleQuestions: true`)
- 말문제(소카테고리 2차 선택 가능) / 표지 및 상황문제 / 동영상 문제 / 랜덤 전체.
- 문제마다 선택 즉시 채점·해설 공개.

### 8. 즐겨찾기 & 오답 노트
- 퀴즈 중 별 아이콘으로 즐겨찾기 토글.
- 틀린 문항은 자동으로 오답 노트에 누적, 다시 맞히면 제거.

### 9. 문항별 누적 통계 (`UserAnswerStatsService`)
- 시도 수 / 정답 수 / 보기별 선택 횟수 저장.
- "최다 오답 문항 10선" 자동 도출.

### 10. 실격 기준 카탈로그
- `assets/driving_disqualification_merged.json` (도로교통공단 공식)을 홈 하단에 3.5초 간격 랜덤 표시.
- 상세 화면에서 기능시험/도로주행 전체 목록 + 공식 출처 링크.

### 11. 다국어 & 테마
- ko / en / zh / vi UI + 각 언어 전용 문제 은행.
- 시스템 / 라이트 / 다크 테마 전환, 설정은 `SharedPreferences` 로 유지.

## MVP 제외 사항 (의도적 비목표)
- 클라우드 퀴즈 데이터 동기화 — 퀴즈 진도·오답·즐겨찾기·통계는 단말 `SharedPreferences` 에 로컬 저장
- 결제/광고/프리미엄 티어
- 문항 공유, 커뮤니티, 해설 작성
- 실제 문제지 PDF 다운로드 / 오프라인 파일 export
- 음성 읽기(TTS), 접근성 확대 — 시스템 글꼴 스케일만 존중
- 푸시 알림, 학습 리마인더
- 공식 시험 접수 기능 — 대신 도로교통공단 공식 사이트로 `url_launcher` 이동

## 성공 지표 (개인 도구 기준)
- 신규 사용자가 앱 실행 → 동의 완료 → 모의고사 1회 완료까지 7분 이내
- 전체 문제 은행(1,000문항) 한 번 풀이 완주 가능성 — "풀어본 문항" 진도 카드로 추적
- 오답 목록에서 바로 재풀이 가능 (홈 → 문제 풀기 → 오답 노트 → 재시작)
- 학습 토픽에서 핵심 개념·시험 출제 분석을 확인한 뒤, 연습 메뉴에서 소카테고리 단위로 관련 문제 풀이 가능
